## Profile/Community Media Decision Package

### Purpose

This package turns the current `PROFILE COMMUNITY MEDIA CONTRACT` decision scaffold into a final no-code decision package that can be approved or rejected as future canonical truth.

This package does not:

- activate contract truth
- modify `actual_truth/contracts/profile_community_media_contract.md`
- generate tasks
- resume execution
- mutate baseline or implementation
