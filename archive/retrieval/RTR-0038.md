# RTR-0038
task_id: RTR-0038
title: Enforce retrieval startup binding to the active index contract version
domain: retrieval
problem_type: violation
problem_statement: Retrieval startup does not verify that loaded authoritative artifacts match the active contract version and chunk manifest hash.
evidence_sources: [actual_truth/contracts/retrieval/determinism_contract.md:190-218, tools/index/search_code.py:350-351, tools/index/build_vector_index.py:32-35]
impacted_files: [tools/index/search_code.py, tools/index/build_vector_index.py]
current_truth: Retrieval opens stored artifacts without contract-version or chunk-manifest handshake.
emergent_truth: Retrieval can operate on artifacts from a different contract generation.
target_state: Retrieval startup validates `contract_version` and `chunk_manifest_hash` against the active authoritative manifest before serving results.
verification_method: Retrieval stops before query execution when stored artifact version or manifest hash differs from the active canonical manifest.
dependencies: [RTR-0013, RTR-0015, RTR-0031]
launch_impact: critical
